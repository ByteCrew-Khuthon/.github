### 주요 기능
- 돼지 건강에 치명적인 돼지 기침을 관련 데이터가 학습된 AI를 통해 감지하고 푸시 알림을 전송하여 축사 내 가축들의 전염병 확산을 방지해요.
- 질병이 발생하는 지역을 지도로 확인할 수 있어요.
- 축사 자동 배급, 축사 내 이상기온 등을 감지해 푸시 알림으로 확인할 수 있어요.
